Framework


  • Estos incluyen soportes de programas, bibliotecas y lenguajes interpretados para la ayuda del dise;o y desarrollo de componentes.

  • Es una arquitectura de software que modela las relaciones generales de las entidades de dominio, tambien provee una estructura y una metodología especial de trabajo.

Entorno de Desarrollo Integrado - IDE





  • Programa compuesto por un conjunto de herramientas

  • Entorno de programacion que se lo empaqueto como un programa de aplicación.

  • Se lo puede usar exclusivamente con un lenguaje de programacion o con varios

IDE Android Studio



Es un entorno de programación empaquetado como un programa o aplicación, que nos provee de un marco de trabajo agradable para la mayoría de los lenguajes de programación.
Dentro de la aplicacion la actividad o activity es el principal componente que se encarga de manejar una interfaz grafica.





Activity



El primer activity y mas relevante es el MainActivity de clase Java que es el que nos permite controlar las actividades y hacer que se ejecuten las actividades.







El segundo activity o de layout que es un archivo xml donde esta configurado toda la apariencia de los elementos de la interfaz grafica.
En este archivo se puede ir agregando ya sean botones, etiquetas guias, textos, widgets, contenedores para lograr el diseño esperado.




XML


Dentro del xml podemos usar este modo que es el modo code en este se encuentran las configuraciones y diseños que realizamos y aqui se encuentra todo lo que hemos realizado como codigo.





El modo split nos permite visualizar tanto el codigo como el diseño que se realiza en la interfaz de manera simultanea.







Finalmente el modo Design es el que nos permitira visualizar el diseño de la interfaz de nuestra aplicacion.
