  • el primero es de nombre keycon tipoKey

  • el segundo es de nombre title con el tipo de campo this.title y se inicializa automáticamente this.title con el valor pasado.

Luego se crea el estado mutable de este widget en una ubicación determinada del árbol. El framework puede llamar a este método varias veces durante la vida útil de un StatefulWidget

Cada widget en Flutter se crea a partir de un build método, y cada método de construcción toma BuildContextcomo argumento, este retornara un widget
