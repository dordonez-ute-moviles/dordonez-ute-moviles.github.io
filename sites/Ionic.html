Qué es IONIC?

Ionic Framework se centra en la interacción de interfaz de usuario y experiencia de usuario de una aplicación:

  • Controles de interfaz de usuario, interacciones, gestos, animaciones.
  • Es fácil de aprender y se integra con otras bibliotecas o marcos, como Angular , React o Vue.
  • Alternativamente, se puede usar de forma independiente sin ningún marco de interfaz utilizando un script simple include.


    • Ionic está diseñado para funcionar y mostrarse a la perfección en todos los dispositivos y plataformas móviles actuales. Con componentes listos para usar, tipografía y un tema base magnífico (pero extensible) que se adapta a cada plataforma, estará construyendo con estilo.

  • Ionic emula las pautas de la interfaz de usuario de la aplicación nativa y utiliza SDK nativos, lo que reúne los estándares de la interfaz de usuario y las características del dispositivo de las aplicaciones nativas junto con todo el poder y la flexibilidad de la web abierta.
  • Ionic usa Capacitor (o Cordova) para implementar de forma nativa o se ejecuta en el navegador como una aplicación web progresiva.



    • IONIC- Multiplataforma

  • Cree e implemente aplicaciones que funcionen en múltiples plataformas, como iOS nativo, Android, escritorio y la web como una aplicación web progresiva, todo con una base de código. Escribe una vez, corre a cualquier lugar.

    • IONIC- Basado en estándares web

  • Ionic Framework está construido sobre confiables, tecnologías web estandarizadas: HTML, CSS y JavaScript, utilizando API web modernas como Elementos personalizados y Shadow DOM. Debido a esto, los componentes de Ionic tienen una API estable y no están en el capricho de un solo proveedor de plataforma.


  • Limpio, sencillo y funcional. Ionic Framework está diseñado para funcionar y mostrarse maravillosamente listo para usar en todas las plataformas. Comience con componentes prediseñados, tipografía, paradigmas interactivos y un tema base magnífico (pero extensible).

    • IONIC- Compatibilidad

  • Si bien las versiones anteriores de Ionic estaban estrechamente acopladas a Angular, la versión 4.x del marco se rediseñó para que funcione como una biblioteca de componentes web independiente , con integraciones para los últimos marcos de JavaScript, como Angular. Ionic se puede usar en la mayoría de los marcos frontend con éxito, incluidos React y Vue, aunque algunos marcos necesitan una corrección para el soporte completo de componentes web.

    • IONIC- JavaScript

  • Uno de los principales objetivos al trasladar Ionic Framework a Web Components era eliminar cualquier requisito estricto en un solo framework para alojar los componentes. Esto hizo posible que los componentes principales funcionaran de forma independiente en una página web con solo una etiqueta de script. Si bien trabajar con marcos puede ser excelente para equipos y aplicaciones más grandes, ahora es posible usar Ionic como una biblioteca independiente en una sola página, incluso en un contexto como WordPress.

    • IONIC- Angular

  • Angular siempre ha estado en el centro de lo que hace grande a Ionic. Si bien los componentes principales se han escrito para funcionar como una biblioteca de componentes web independiente, el @ionic/angularpaquete hace que la integración con el ecosistema Angular sea muy sencilla. @ionic/angularincluye toda la funcionalidad que los desarrolladores de Angular esperarían de Ionic 2/3 y se integra con las bibliotecas centrales de Angular, como el enrutador Angular.

    • IONIC- Vue

  • Ionic ahora tiene soporte oficial para la popular biblioteca Vue 3. Ionic Vue permite a los desarrolladores de Vue usar sus habilidades web existentes para crear aplicaciones destinadas a iOS, Android, la web y el escritorio. Con @ionic/vue, puede usar todos los componentes principales de Ionic, pero de una manera que se siente como si usara componentes nativos de Vue.

    • IONIC- React

  • Ionic ahora tiene soporte oficial para la popular biblioteca React. Ionic React permite a los desarrolladores de React usar sus habilidades web existentes para crear aplicaciones dirigidas a iOS, Android, la web y el escritorio. Con @ionic/react, puede usar todos los componentes principales de Ionic, pero de una manera que se siente como usar componentes nativos de React.
