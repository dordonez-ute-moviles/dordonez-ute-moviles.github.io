Qué es OsmDroid?

  • La biblioteca “osmdroid” de Android que proporciona herramientas y vistas para interactuar con OpenStreetMap-Data.
  • OpenStreetMapView es un reemplazo (casi) completo y gratuito para la clase MapView de Android.

  • MapView de osmdroid es básicamente un reemplazo de la clase MapView de Google

    Soporte WMS

    Requisitos generales del servidor WMS

  • El servidor WMS debe admitir la producción de un mosaico para un cuadro delimitador específico con un tamaño de píxel específico.
  • No se admiten capas que requieren parámetros de entrada adicionales, como desplazamientos de tiempo./li>
  • No se admiten las capas que requieren un mensaje de publicación HTTP.
  • No se admiten las capas que utilizan píxeles no cuadrados (es decir, la altura y el ancho de la imagen no son iguales).
  • Solo se pueden usar las capas que usan el CRS / SRS compatible
  • Se admite el estilo por capa, el primer estilo se selecciona automáticamente.
  • Actualmente, no se admiten varias capas de mosaicos activos al mismo tiempo (esto sería útil para obtener el mejor mosaico disponible al acercar y alejar).
