Bluetooth

14 de Enero de 2021

Bluetooh en Android

Utiliza el protocolo RFCOMM

  • Radio Frequency COMMunication
  • Emula un puerto serial
  • Similar a TCP: usa sockets
  • Un dispositivo puede establacer conexión con hasta 60 dispositivos simultaneamente (depende de la capa de android)
    • Cada conexión usa su propio canal independiente
  • Un dispositivo es servidor y otro cliente
  • El servidor abre un socket y queda a la escucha
  • El cliente no puede Recibir mas clientes por dicho socket
    • Pero puede abrir otros sockets para recibir mas clientes
      • O conectarse como cliente con otros dispositivos

Actividades pricipales

  • Configuración
  • Busqueda
  • Conexión
  • Tansferencia de datos

Permisos

Para usar bluetooth se debe incluir en el manifest

  • -android.permission.BLUETOOTH
  • -android.permission.BLUETOOTH_ADMIN

Este último solo es necesario para búsqueda y configuración

Es muy recomendable tambien incluir

  • -android.permission.ACCESS_FINE_LOCATION

Adaptador Bluetooth

Para usar bluettoth es necesario tomar una referencia al adaptador.

  • BluetoothAdapter btAdapter=
  • BluetoothAdapter.getDefaultAdapter();

Si la referencia devuelta es null no se puede usar bluetooth en el dispositivo

  • btAdapter == null

Adaptador Activo

Verificar si está activo

  • btAdapter.isEnabled()

Caso contrario solicitar activación

  • Intent intetn = new
  • Intent(BluetoothAdapter.ACTION_REQUEST_ENABLE);
  • startActivityForResult(intent, REQUEST_ENABLE_BT);

Un cuadro de diálogo le pedira permiso al usuario

Si se activa, onActivityResult() recibirá

  • RESULT_OK

Caso contrario recibirá

  • RESULT_CANCEL
